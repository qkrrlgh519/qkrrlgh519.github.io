# Java - Basic1

## Hello Java

```java
public class HelloJava {
	public static void main(String[] args) {
		System.out.println("Hello Java");
	}
}
```

- 파일명과 클래스의 이름은 같아야 한다.
- `{}` 블록을 사용해서 클래스, 메서드의 시작과 끝을 나타낸다.

## 참고 : 괄호

- 소괄호 : `()`
- 중괄호 : `{}`
- 대괄호 : `[]`
